# Warp 暮光主题套装

5 套从「一种光」出发的 Warp 配色主题，配你的日落天鹅湖壁纸。

| 主题 | 光的来源 | 强调色 |
|:--|:--|:--|
| `twilight` 暮光 | 日落天鹅湖（你的壁纸） | 琥珀 `#ffb066` + 冷青 |
| `aurora` 极光 | 北境极光 | 青绿 `#7af0c0` + 薰衣草紫 |
| `abyss` 深海 | 深海冷光 | 电蓝 `#5cc8ff` + 青 |
| `sakura` 樱绯 | 暮樱 | 樱粉 `#ff9ec4` + 暖橙 |
| `ember` 余烬 | 篝火余温 | 橙红 `#ff7a59` + 金 |

这 5 个 `.yaml` 由 `../preview.html` 里的**同一套单色算法**生成——预览什么色，导出就是什么色。想要自己的色，见下面「自定义」。

## 自定义：选任意颜色生成主题

打开 `../preview.html`，右下角控制条：

1. 点彩虹环取色器 / 直接输 hex（如 `#9b5de5`），整个界面实时重算
2. 满意后点 **「导出 Warp」** → 浏览器下载一个 `.yaml`
3. `mv ~/Downloads/<文件>.yaml ~/.warp/themes/`，回 Warp 重选即可

算法只要一个种子色，就用色彩理论派生整套：强调色、日落壁纸的冷暖渐变、以及终端 16 色。其中 **red/green/blue 等 ANSI 语义色会锚定到标准色相**（只被种子色轻染 12%），保证 `git diff`、`ls` 在任何主题下都还是「红就是红、绿就是绿」。

## 核心原则：透明只让一层做

> Warp 的窗口透明 + 模糊是**系统级**的，原生、好看、不卡。
> 主题文件**只管颜色**，不碰透明/模糊。两者分开，零叠加。

千万别在主题里再去模拟透明——那只会和 Warp 的窗口模糊互相打架，导致双重模糊、颜色发灰发暗。

## 安装

```bash
mkdir -p ~/.warp/themes
cp ~/Documents/AI/warp-themes/*.yaml ~/.warp/themes/
```

然后重启 Warp（或 `⌘,` 打开设置）：

1. **选颜色**：`Settings → Appearance → Themes` → 选 `Twilight 暮光`（或其他四套）
2. **设透明 + 模糊**：`Settings → Appearance → Window`
   - `Window opacity` 拉到 **80–88%**（壁纸透出来，但正文还清楚）
   - 打开 `Window blur`，**5–15** 之间按口味调
3. 想换主题随时回第 1 步重选——透明/模糊设一次就够，不用动。

## 调参建议

- **想更透**：opacity 往 75% 走，但建议同时把 blur 调高一点，否则壁纸细节会干扰正文。
- **觉得发灰**：多半是 opacity 太低 + blur 太低。先把 blur 提上去，比单纯压暗背景更干净。
- **白天/晚上**：白天用 `abyss` 或 `ember`（对比强），晚上用 `twilight` 或 `sakura`（柔和）。

## 微调颜色

每个 `.yaml` 里 `accent` 是强调色（光标、选中、链接），`background`/`foreground` 是底色和正文。
改完存盘，在 Warp 里重选一次该主题即可生效。
