Argo 1.0.6

Download

This release brings the new Dynamic Island workflow together with the public website, GitHub Pages delivery, and the updated Homebrew cask path.

New

  • Dynamic Island session center: added richer agent session models, expanded/collapsed Island views, closed-agent grids, pane state tracking, and tests around session presentation.
  • Claude hook bridge: added hook installation, notification bridging, interaction routing, and protocol updates so Claude activity can surface inside Argo.

Release

  • Public website and install path: added the website, GitHub Pages deployment, direct DMG download, and the brew install --cask krystal1110/argo/argo cask command.
  • Release packaging: bumped Argo to 1.0.6, refreshed the Sparkle appcast, and published Argo-1.0.6.dmg.
