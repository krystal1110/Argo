整个路径只围绕一件事：把本机会话带到手机上。The flow is simple: carry the local session to your phone.

1Argo workspace
2HAPI Hub Host

从 Argo 到手机预览。From Argo to mobile preview.

先在 Argo 里选中 workspace，再通过 HAPI 菜单启动服务、登录手机端，最后正常和 Claude 交流。Select the workspace in Argo, start the HAPI service, sign in from mobile, then talk to Claude as usual.

1

打开 HAPI 菜单Open HAPI Menu

先选中 Argo workspace，再点顶部网格按钮，在菜单里选择 Open HAPI Menu。Select an Argo workspace, click the top grid button, then choose Open HAPI Menu.

2

启动 Hub 并复制 HostStart Hub and copy Host

选择 HAPI Hub 或 relay 入口，复制生成的 Host，在手机浏览器里打开。Choose HAPI Hub or the relay entry, copy the generated Host, and open it in your phone browser.

3

从 Settings 获取 tokenGet the token from Settings

进入 HAPI 页面里的 HAPI Show Settings，复制 token，然后用它登录手机上的 HAPI 页面。Open HAPI Show Settings, copy the token, then use it to sign in on the phone.

4

启动 HAPI ClaudeStart HAPI Claude

回到 Argo 的 HAPI 菜单，选择 HAPI Claude，让手机连接当前 workspace 的 Claude session。Return to the Argo HAPI menu, choose HAPI Claude, and connect the phone to this workspace session.

5

正常交流并查看 HTMLChat and view HTML

在手机上正常描述你要改的页面，让 Claude 启动预览服务，再打开它给出的 HTML preview。Describe your page changes from the phone, ask Claude to start the preview server, then open the HTML preview it returns.

使用时注意这几件事。A few practical notes.

  • Host 是手机访问 HAPI 的入口，优先复制 HAPI Hub 页面里展示的地址。Host is the mobile entry point. Copy the address shown by HAPI Hub.
  • token 用来登录你的 HAPI 页面，不要发给不信任的人。The token signs in to your HAPI page, so do not share it with untrusted people.
  • HTML preview 由 Claude 在当前 workspace 里启动，手机只需要打开它返回的链接。Claude starts the HTML preview in the current workspace; the phone only opens the returned link.
